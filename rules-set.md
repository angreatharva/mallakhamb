------------------------
1.12 TIE BREAK RULES
------------------------

APPLICABILITY:
- Tie break rules apply ONLY for:
  * All Round Individual Championship
  * Apparatus Championship
- Tie break rules DO NOT apply to Team Championship.

STANDARD TIE BREAK PROCEDURE:

RULE TB-1: If two or more athletes are tied →
           Calculate BASE SCORE.

RULE TB-2: If Base Score is also equal →
           Use Execution Average
           (average of middle execution judges).

RULE TB-3: If still tied →
           (Execution Average + Chief Judge Score) / 2.

RULE TB-4: If still tied →
           Decide by Lucky Draw (Chit).

SPECIAL TIE BREAK – JUDGE PANEL VARIATIONS:

CASE A: 3 JUDGES PANEL
(1 Superior Judge + 2 Normal Judges)

RULE TB-5: If tie persists →
           Compare execution TIME.
RULE TB-6: Athlete with LESSER execution time
           will be ranked higher.
RULE TB-7: If execution time is also equal →
           Superior Judge decision is FINAL.

CASE B: 4 JUDGES PANEL
(1 Superior Judge + 3 Normal Judges)

RULE TB-8: First preference →
           Senior / Chief Judge score.
RULE TB-9: If Chief Judge score is also equal →
           Consider HIGHEST score
           among all judges.
RULE TB-10: If still tied →
            Decide by Lucky Draw (Chit).

############################################################
SECTION 2 – JUDGING, SCORING & DEDUCTIONS
############################################################

------------------------
2.1 JUDGING PANEL
------------------------

RULE JP-1: Panel consists of 1 Chief Judge + 4 Execution Judges.
RULE JP-2: Maximum score per set = 10.00.
RULE JP-3: Chief Judge score excluded from average
           unless Base Score is applied.
RULE JP-4: Judges must submit score slips within 20 seconds.
RULE JP-5: Chief Judge must write score FIRST.
RULE JP-6: Final score must be declared before next athlete starts.

------------------------
2.1.1 TIME MANAGEMENT
------------------------

RULE TM-1: Timer starts when athlete touches apparatus.
RULE TM-2: Warning whistle at 60 seconds.
RULE TM-3: Final whistle at 90 seconds.
RULE TM-4: Performance beyond 90 seconds attracts time deduction.

------------------------
2.1.2 AVERAGE SCORE CALCULATION
------------------------

RULE AS-1: Sort 4 execution judge scores.
RULE AS-2: Remove highest and lowest scores.
RULE AS-3: Average remaining two = Execution Average.

------------------------
2.1.3 TOLERANCE & BASE SCORE
------------------------

RULE BS-1: Compare Execution Average with Chief Judge score.
RULE BS-2: If difference is within tolerance →
           Execution Average is final.
RULE BS-3: If difference exceeds tolerance →
           Base Score is mandatory.
RULE BS-4: Base Score =
           (Execution Average + Chief Judge Score) / 2.

TOLERANCE TABLE:
- 9.00 – 10.00 : 0.10
- 8.00 – 8.99  : 0.20
- 7.00 – 7.99  : 0.30
- 6.00 – 6.99  : 0.40
- 5.00 – 5.99  : 0.50
- Below 5.00   : 1.00

------------------------
2.2 SCORE DISTRIBUTION (MAX 10.00)
------------------------

RULE SD-1: Difficulty  = 3.80
RULE SD-2: Combination = 1.60
RULE SD-3: Execution   = 4.40
RULE SD-4: Originality = 0.20

------------------------
2.2.1 DIFFICULTY VALUES
------------------------

RULE DF-1: A Class = 0.20
RULE DF-2: B Class = 0.40
RULE DF-3: C Class = 0.60

REQUIRED ELEMENTS:
- TEAM: A=4, B=6, C=1
- INDIVIDUAL / APPARATUS: A=1, B=6, C=2

RULE DF-4: Total Difficulty must equal 3.80.

------------------------
2.2.2 COMBINATION RULES
------------------------

RULE CB-1: Full apparatus utilization required.
RULE CB-2: Right & left side execution required.
RULE CB-3: Forward & backward flexibility required.
RULE CB-4: Minimum element count mandatory.
RULE CB-5: Missing combination items result in deduction from 1.60.

------------------------
2.2.3 EXECUTION CRITERIA
------------------------

RULE EX-1: Proper technique required.
RULE EX-2: Correct posture mandatory.
RULE EX-3: Controlled movement mandatory.
RULE EX-4: Proper amplitude required.
RULE EX-5: Clean catches required.
RULE EX-6: Stable landing required.
RULE EX-7: Continuity and speed considered.

------------------------
2.2.4 ORIGINALITY BONUS
------------------------

RULE OB-1: Maximum originality bonus = 0.20.
RULE OB-2: Element must be pre-approved.
RULE OB-3: Approval valid for 365 days.

------------------------
2.3 FINAL SCORE DEDUCTIONS
------------------------

- Not reporting to Chief Judge : -0.10
- Wrong sequence              : -0.20
- Late start (>10 sec)        : -0.20
- Improper attire             : -0.20
- Team not uniform            : -0.50
- Misbehavior                 : -1.00
- Gross misconduct            : Disqualification possible

------------------------
2.3.1 TIME DEDUCTIONS (>90 sec)
------------------------

- Up to 3 sec   : -0.20
- Up to 5 sec   : -0.30
- Up to 9 sec   : -0.40
- 10 sec or more: -0.50

------------------------
FINAL SCORE FORMULA
------------------------

FINAL_SCORE =
Difficulty
+ Combination (after penalties)
+ Execution (after deductions)
+ Originality
- Time deductions
- Final deductions

MAX FINAL SCORE = 10.00
MIN FINAL SCORE = 0.00
